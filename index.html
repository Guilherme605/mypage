Projetos

DelitosMaps é um projeto em desenvolvimento com o objetivo de mapear os crimes e servir de repositório informativo para a comunidade de Planaltina-DF.
Trata-se do meu projeto de TCC para o Instituto Federal de Ciência e Tecnologia de Brasília.

Reactjs

Para tornar as interfaces em SPA, foi utilizado o React, bem como componentizar suas features como cabeçalho, footer e paginação, reutilizando código e trazendo maior performance na aplicação.

java - spring

Java foi a linguagem de desenvolvimento do BackEnd, com auxílio do framework Spring Boot, foi efetuado a API, em padrão de projetos em camadas,(DTO, repository, service, controller, model), integração com banco de dados.

testes

Como uma das boas práticas, é tendência executar e documentar os testes das funcionalidades. Nesse projeto foi utilizado o Junit para executar essa testagem em seus repositórios na camada de acesso a dados.

documentação
uml

Durante todo o desenvolvimento cada etapa do processo foi documentada utilizando princípios de engenharia de software, bem como os diagramas UML necessários para um entendimento e manutenção do sistema.
